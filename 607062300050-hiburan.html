Heboh Petasan di Tempat Latihan Timnas, Ternyata Sifatnya Hiburan

Sumber Gambar : Detik.com

HANOI, iNews.id - Heboh pesatan di tempat latihan Timnas Indonesia di Hanoi. Ternyata itu berasal dari pasar di sekitar hotel dan sifatnya hiburan.

Skuad Garuda bertamu ke markas Vietnam di Stadion My Dinh pada matchday keempat Grup F Kualifikasi Piala Dunia 2026 Zona Asia putaran kedua, Selasa (26/3/2024). Jelang laga, heboh penampakan kembang api di tempat latihan Timnas Indonesia.

Awalnya netizen Tanah Air berspekulasi jika kembang api tersebut bermaksud untuk meneror para pemain Timnas Indonesia. Sebab sebelumnya oknum suporter Garuda melakukan hal tersebut saat Vietnam berkunjung ke Jakarta.

Namun faktanya PSSI mengonfirmasi jika kembang api tersebut berasal dari pasar. Mereka memastikan suasana hotel dan tempat latihan Timnas Indonesia sangat kondusif.

Sayangnya skuad Garuda akan tampil pincang melawan Vietnam nanti. Beberapa pemain menderita demam jelang pertandingan.

Pelatih Timnas Indonesia Shin Tae-yong mengonfirmasi kondisi beberapa pemainnya cukup parah. Kemudian ada tiga penggawa yang dipastikan absen yakni Nadeo Argawinata, Marc Klok dan Sandy Walsh.

Nadeo dan Klok dicoret dari skuad karena cedera. Sedangkan Sandy Walsh diparkir karena akumulasi kartu kuning.
